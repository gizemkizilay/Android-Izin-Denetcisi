# 🧩 İzin Analiz Modülü
Bu modül, Python yerleşik `xml.etree.ElementTree` kütüphanesini kullanarak parçalanan `AndroidManifest.xml` dosyası içindeki `uses-permission` etiketlerini ayıklar. Tespit edilen izinleri, sistemde tanımlı "Tehlikeli İzinler" veritabanı ile karşılaştırarak uygulamanın risk profilini çıkarır.
